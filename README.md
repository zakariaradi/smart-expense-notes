# 💰 Smart Expense Notes

Smart Expense Notes is a simple and intuitive web application that allows users to track their daily expenses using natural text notes.

Instead of filling out traditional forms, users can simply type something like:

Coffee 25  
Uber to work 18  
Dinner pizza 60  

The application automatically extracts the amount and displays the expense.

---

## 🚀 Tech Stack

- React (Vite)
- Tailwind CSS
- JavaScript
- Local Storage (coming soon)

---

## ✅ Current Features

- Add expense using a single text input
- Automatically extract amount from note
- Display expense list dynamically
- Basic validation
